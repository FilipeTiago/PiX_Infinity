  • Open Virtual Keyboard Settings

  • How to use:
    Important! To start using it, press the Start button first. After each page reload, do it again.

    Auto:
    - If you choose to Auto, the on-screen keyboard will automatically turn on when you enter input or form, etc on the web page.
    On Demand:
    - If you choose On Demand, a small icon will appear every time you want to type text, click on that small icon to bring up the virtual keyboard.
    Always On:
    - On-screen keyboard will always be enabled on your website. But after importing data, it will be hidden automatically
